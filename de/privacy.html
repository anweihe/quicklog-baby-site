Übersicht

Der Schutz Ihrer personenbezogenen Daten ist uns ein wichtiges Anliegen. Quicklog Baby wurde so entwickelt, dass Ihre Privatsphäre an erster Stelle steht. Alle Daten werden ausschließlich lokal auf Ihrem Gerät gespeichert. Es werden keine personenbezogenen Daten an Server übertragen.

Datenspeicherung

Sämtliche Tracking-Daten (Fütterungen, Wickeln, Schlaf usw.) werden in einer lokalen SQLite-Datenbank auf Ihrem Gerät gespeichert. Der Entwickler hat keinen Zugriff auf diese Daten.

Keine Datenerhebung

  • Es werden keine personenbezogenen Daten durch den Entwickler erhoben.
  • Es werden keine Daten an Server oder Dritte übermittelt.
  • Es werden keine Analyse- oder Tracking-Tools eingesetzt.
  • Diese Website verwendet keine Cookies.
  • Es werden keine Drittanbieter-Dienste oder SDKs integriert.
